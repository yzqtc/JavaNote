变量分类

  • 变量分类概念
  • 1、局部变量(local variable)
  •  方法或语句块内部定义的变量。生命周期是从声明位置开始到方法或语句块执行完毕为止。局部变量在使用前必须先声明、初始化(赋初值)再使用
  •  局部变量声明:

public void test(){
int i;
int j = i+5;//编译出错，变量i还未被初始化
}
public void test(){
int i;
i = 10;
int j = i+5;//编译正确
}
复制
  • 2.成员变量(也叫实例变量 member variable)
  • 方法外部类的内部定义的变量。从属于对象，生命周期伴随对象终止。如果不自行初始化，它会自行初始化成该类型的默认初始值
  • 实例变量默认初始值
    数据类型 实始值
    int 0
    double 0.0
    char '\u0000'
    boolean false
  • 实例变量声明:
1
2
3
public class Test{
int i;
}
复制
  • 3.静态变量(类变量 static variable)
  • 使用static定义。从属于类，生命周期伴随类终止，从类加载到卸载。如果不自行初始化，与成员变量相同会自动初始化成该类型的默认初始值
  • 静态变量的声明:
1
static int size;
复制
  • 局部变量、成员变量、静态变量的区别
  • 局部变量、成员变量、静态变量的区别
    类型 声明的位置 从属于 生命周期
    局部变量 方法或语句块内部 方法/语句块 从声明位置开始，直到方法或语句块执行完毕，局部变量消失
    成员变量(实例变量) 类内部，方法外部 对象 对象创建，成员变量也跟着创建。对象消失，成员变量也跟着消失
    静态变量(类变量) 类内部，static修饰 类 类被加载，静态变量就有效；类别卸载，静态变量消失

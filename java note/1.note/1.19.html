  • [示例1]if单选择结构:
  • 1 public class Test1 {
    2 public static void main(String[] args) {
    3 //通过掷三个骰子看看今天的手气如何？
    4 int i = (int)(6 * Math.random()) + 1;//通过Math.random()产生随机数

    6 int k = (int)(6 * Math.random()) + 1;
    7 int count = i + j + k;
    8 //如果三个骰子之和大于15，则手气不错
    9 if(count > 15) {
    10 System.out.println("今天手气不错");
    11 }
    12 //如果三个骰子之和在10到15之间，则手气一般
    13 if(count >= 10 && count <= 15) { //错误写法：10<=count<=15
    14 System.out.println("今天手气很一般");
    15 }
    16 //如果三个骰子之和小于10，则手气不怎么样
    17 if(count < 10) {
    18 System.out.println("今天手气不怎么样");
    19 }
    20 System.out.println("得了" + count + "分");
    21 }
    22 }

    2 语句块1
    3 }else{

  • [示例2]if-else结构:
  • 1 public class Test2 {
    2 public static void main(String[] args) {
    3 //随机产生一个[0.0, 4.0)区间的半径，并根据半径求圆的面积和周长
    4 double r = 4 * Math.random();
    5 //Math.pow(r, 2)求半径r的平方
    6 double area = Math.PI * Math.pow(r, 2);
    7 double circle = 2 * Math.PI * r;
    8 System.out.println("半径为： " + r);
    9 System.out.println("面积为： " + area);
    10 System.out.println("周长为： " + circle);
    11 //如果面积>=周长，则输出"面积大于等于周长"，否则，输出周长大于面积
    12 if(area >= circle) {
    13 System.out.println("面积大于等于周长");
    14 } else {
    15 System.out.println("周长大于面积");
    16 }
    17 }
    18 }

  • if-else if-else多选择结构
  • 语法结构:
  • 1 if(布尔表达式1){
    2 语句块1
    3 } else if(布尔表达式2) {
    4 语句块2
    5 }……
    6 else if(布尔表达式n){
    7 语句块n
    8 } else {
    9 语句块n+1
    10 }
  • 当布尔表达式1为真时，执行语句块1;否则，判断布尔表达式2，当布尔表达式2为真时，执行语句块2;否则，继续判断布尔表达式3······;如果1~n个布尔表达式均判定为假时，则执行语句块n+1，也就是else部分。流程图如图所示

  • switch多选择结构
  • 语法结构:
  • 1 switch (表达式) {
    2 case 值1:
    3 语句序列1
    4 [break]
    5 case 值2:
    6 语句序列2
    7 [break]
    8 … …
    9 [default:
    10 默认语句]
    11 }
  • switch语句会根据表达式的值从相匹配的case标签处开始执行，一直执行到break语句处或者是switch语句的末尾。如果表达式的值与任一case值不匹配，则进入default语句(如果存在default语句的情况)
  • 根据表达式值的不同可以执行许多不同的操作。switch语句中case标签在JDK1.5之前必须是整数(long类型除外)或者枚举，不能是字符串，在JDK1.7之后允许使用字符串(String)
  • 需要注意的是，当布尔表达式是等值判断的情况，可以使用if-else if-else多选择结构或者switch结构，如果布尔表达式区间判断的情况，则只能使用if-else if-else多选择结构

    19 default:

    21 }
    22 }
    23 }
